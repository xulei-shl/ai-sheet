# AI-Sheet 使用说明

## 什么是 AI-Sheet

AI-Sheet 是一款桌面应用，把 AI 对话与 Excel 处理深度结合。你直接用自然语言描述需求，AI 就能读取你表格里的数据，自动生成公式、编写处理提示词、甚至直接写出 Python 脚本并执行。

**核心流程：**

1. 先加载 Excel 文件，选择要处理的 Sheet 和列。
2. 这些文件、表格、列名`加载`到 AI 对话的上下文
3. 然后选择处理方式：生成公式、生成批量处理提示词、或让 AI 编写 Python 脚本自动执行。

---

## 界面概览

```
左侧          中间              右侧
导航          数据/操作页面      AI 对话面板
(可折叠)      （公式、AI、       （跨页面持续存在，
             配置等）           始终与当前数据联动）
```

- **左侧**：Tab 导航，包括数据、公式、AI、配置、技能管理。
- **中间**：对应页面的操作区，完成公式编辑、批量处理配置等。
- **右侧**：AI 对话面板，始终可用。它自动知道当前加载了哪些文件、哪些 Sheet、哪些列被选中。

---

## 第一步：加载数据并进入上下文

1. 点击左侧「数据」页签。
2. 把 Excel 文件拖进窗口，或使用上传按钮导入。
3. 每个文件会自动展开，显示其中所有 Sheet。
4. 切换到「公式」「AI」「LLM 批量处理」等页面后，左侧列表继续显示这些文件，你随时可以在不同页面重新选择 Sheet。

**上下文机制：** 一旦你选中某个 Sheet（有时还选了几列），这些信息`加载`到右侧 AI 对话**。你可以在对话里直接说"把 A 列和 B 列相加"，AI 已经知道 A、B 是什么。

---

## 处理方式一：公式——AI 写公式，直接写入表格

适合数学计算、文本拼接、条件判断、查找引用等 Excel 原生公式可处理的任务。

**操作步骤：**

1. 点击左侧「公式」。
2. 在左侧文件树点击选中目标 Sheet。
3. 在右上方选择**目标列**：可以覆盖现有列，或新建一列存放结果。
4. **需求输入**：你可以直接描述，例如：
   - "计算金额列与数量列的乘积"
   - "从邮箱列中提取域名"
   - "当状态列为空时显示待处理"
5. AI 会生成可用公式。你也可以直接在中间编辑框中手写。
6. 点击「预览前 3 行」查看效果。
7. 确认无误后点击「应用公式」，整列自动填充。

**快捷公式收藏：**

- 应用过的公式会被记入「历史记录」。
- 点击历史旁的图钉图标，给公式起个名字（如"金额×数量"），固定在左侧底部。
- 下次直接点击快捷公式名称，公式自动填入编辑框。
- 再次点击则可取消固定。

---

## 处理方式二：大模型批量处理——调用 AI 处理数据

适合翻译、分类、摘要、信息提取、内容改写等需要 AI 理解语义的任务。

**操作步骤：**

1. 点击左侧「AI → LLM 批量处理」。
2. 在左侧文件树选中需要处理的文件和 Sheet。
3. 在中间配置区完成设置：
   - **选择模型：** 从已有模型配置中选择（需先在「配置」页添加）。
   - **选择输入列：** 勾选要用作 AI 处理依据的列（可多选）。
   - **选择输出列：** 选择已存在的列覆盖写入，或选择「[新建列...]」输入新列名。
   - **错误信息列：** 自行命名一列，用于记录处理失败的信息。
   - **批次大小：** 控制并发请求数量（建议 3~5）。
   - **提示词：** 两种方式——选取「已保存」的模板，或直接「自定义」输入本轮需求。
4. 点击「开始」，系统把输入列内容发送给大模型，结果再写回指定列。
5. 右下角终端实时显示每行处理状态与返回内容。

**提示词模板使用：**

- 左侧「提示词」页面可以新建、编辑、搜索模板。
- 编写提示词时，用 `{列名}` 引用数据，例如：
  ```
  把以下产品名称翻译成英文：
  {产品名称}
  ```
- 保存后在批量处理页面直接选用，无需重复输入。

---

## 处理方式三：Python 脚本——AI 编写并自动执行

适合规则复杂、多步骤的数据清洗、跨文件合并、统计汇总、生成报表等情境。

**操作步骤：**

1. 先在「数据」页面加载 Excel。
2. 在右侧 AI 对话面板中直接描述需求，例如：
   - "读取 pricing.xlsx，筛选销量大于 1000 的行，另存为 top_products.xlsx"
   - "按省份分组，计算每位客户的总消费"
   - "合并本月三个销售表，去重后加上序号"
3. AI 会自动编写 Python 脚本并执行。
4. 如果运行出错，AI 会自动诊断并修改脚本后重试，直到成功。

> 快捷入口：左侧「AI」页面也有「Python 处理」卡片，点击后对话框会自动填入引导语，方便快速启动。

---

## AI 内置技能

AI-Sheet 随 Agent 带两个内置技能：

| 技能 | 说明 |
|------|------|
| **Python 处理** | 规范化的 Python 数据处理工作流，包含 pandas、openpyxl 等常用库使用约定。在对话里输入 `/skill:python-processing` 可显式激活。 |
| **自我提升** | AI 可主动学习本次处理经验，优化后续对话。 |

技能可通过左侧「技能管理」页面浏览和编辑，也支持从本地文件夹导入新技能。

---

## 快速上手建议

1. **首次使用：** 先到「配置」添加你自己的 LLM 模型（需要 API Key，加密保存在本地）。
2. **最简单的测试：** 加载 Excel → 进入「公式」→ 让 AI 生成一个简单的拼接或计算 → 预览 → 应用。
3. **批量语义任务：** 把常用需求整理成提示词模板保存 → 用「LLM 批量处理」直接套用。
4. **复杂任务交给 Python：** 用自然语言描述完整需求给 AI，让它写脚本并自动修错跑通。
