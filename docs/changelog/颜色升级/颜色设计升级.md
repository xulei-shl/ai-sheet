# 黑色系主题

## 1. 颜色配置 (`style/Color/Wilderness.css`)

该主题仅应用于 **暗色模式**，需在 `[data-theme-mode="dark"]` 选择器下使用：

```css
[data-theme-mode="dark"] {
  /* === 主色 === */
  --QYL-custom-primary-main: 123deg;
  --b3-theme-primary: #9db56d;             /* 橄榄绿色主色 */
  --b3-theme-primary-light: rgba(175, 214, 98, 0.5);
  --b3-theme-primary-lighter: rgba(175, 214, 98, 0.3);
  --b3-theme-primary-lightest: rgba(175, 214, 98, 0.12);

  /* === 编辑器背景 === */
  --b3-theme-background: #2d353b;          /* 深灰绿主背景 */
  --b3-theme-background-light: #465057;    /* 浅一级背景 */

  /* === 面板/菜单层 === */
  --b3-theme-surface: #232a2e;             /* 面板底色 */
  --b3-theme-surface-light: rgba(35, 42, 46, 0.8);
  --b3-theme-surface-lighter: rgba(189, 229, 255, 0.15);

  /* === 文字颜色（主要可读性）=== */
  --b3-theme-on-primary: rgba(255,255,255,.9);
  --b3-theme-on-error: var(--b3-theme-on-primary);
  --b3-theme-on-secondary: var(--b3-theme-on-primary);
  --b3-theme-on-background: #ece4d0;       /* 暖米白（最亮文字） */
  --b3-theme-on-surface: #dfd6bf;
  --b3-theme-on-surface-light: #d1c8b4;

  /* === 顶部工具栏 === */
  --b3-toolbar-blur-background: #444e54;
  --b3-toolbar-color: #d1c8b4;
  --b3-toolbar-hover: rgba(115, 132, 81, 0.5);

  /* === 滚动条 === */
  --b3-scroll-color: #4f5a67;

  /* === 列表 / 悬停 === */
  --b3-list-hover: rgba(115, 132, 81, 0.5);
  --b3-list-icon-hover: rgba(115, 132, 81, 0.5);

  /* === 菜单 / 提示 === */
  --b3-menu-background: #2d353b;
  --b3-tooltips-background: #2d353b;
  --b3-tooltips-color: #dfd6bf;
  --b3-tooltips-second-color: #d1c8b4;
  --b3-tooltips-shadow: 0px 13px 25px -2px 
    rgba(175, 214, 98, 0.12), 0px 0px 10px 0px 
    rgba(175, 214, 98, 0.12);

  /* === 开关 === */
  --b3-switch-background: #54636d;

  /* === 代码块 === */
  --b3-protyle-code-background: #232a2e;

  /* === 页签（File Tab）=== */
  --QYL-tab-item: rgba(158, 195, 83, 0.15);
  --QYL-tab-item-focus: rgba(158, 195, 83, 0.35);

  /* === 毛玻璃效果 === */
  --QYL-Aero-background: rgba(34, 39, 38, 0.7);
  --QYL-Aero-background-wrap: rgba(56, 65, 65, 0.7);

  /* === 引述块 === */
  --QYL-blockquote-background: #3d474d;
  --QYL-blockquote-svg: #515c63;
  --QYL-blockquote: #9ca6ac;
}
```

## 2. 字体配置系统

项目中通过 `script/element/CustomFontStyle.js` 维护，存储为 JSON（`QYL-CustomFontStyle.json`）。该主题支持 **13 级**字体颜色和背景自定义插槽：

| Variable Range | 用途 |
|---------------|------|
| `--b3-font-color1` ~ `--b3-font-color13` | 字体前景色（按色阶分配） |
| `--b3-font-background1` ~ `--b3-font-background13` | 字体背景色（高亮） |

存储配置格式示例（仅列出 light 侧，dark 侧同理）：
```json
{
  "fontcolor1-light": "#hex",
  "cssfontcolor1-light": "optional inline css",
  "fontbackground1-light": "#hex",
  "cssfontbackground1-light": "optional inline css",
  "...": "重复到 slot 13"
}
```

## 4. 在其他项目中引用

推荐做法（Web/前端项目）：

```css
/* 强制暗色模式应用 */
[data-theme-mode="dark"] {
  /* 粘贴上述所有变量 */
}

/* 或改为普通 class，不依赖 data 属性 */
.dark-mode {
  /* 同上 */
}
```

### 可直接复制的调色板

| 语义 | 变量 | HEX / RGBA |
|------|------|-----------|
| 主色 | `--b3-theme-primary` | `#9db56d` |
| 强背景 | `--b3-theme-background` | `#2d353b` |
| 面板背景 | `--b3-theme-surface` | `#232a2e` |
| 最亮文字 | `--b3-theme-on-background` | `#ece4d0` |
| 标准文字 | `--b3-theme-on-surface` | `#dfd6bf` |
| 次要文字 | `--b3-theme-on-surface-light` | `#d1c8b4` |
| 工具栏背景 | `--b3-toolbar-blur-background` | `#444e54` |
| 代码块背景 | `--b3-protyle-code-background` | `#232a2e` |
| 悬停 | `--b3-list-hover` | `rgba(115,132,81,0.5)` |

### 核心风格特点
- **主色**为低饱和橄榄绿（`#9db56d`），饱和度约 25%，适合需要“野外/自然感”的暗色主题
- 文字全部使用暖色系（`#dfd6bf`、`#ece4d0`），避免纯白，降低对比度疲劳
- 大量使用主色的低透明度 rgba（12%~50%）实现层级感
- 引用：`Wilderness.css`、`Wildness.js`

### 3. JS 注入机制 (`script/color/WoodAsh.js`)

```js
export function initWoodAsh() {
  const link = document.createElement('link');
  link.id = 'QYL-WoodAsh';
  link.rel = 'stylesheet';
  link.href = '/appearance/themes/QYL-theme/style/Color/WoodAsh.css';
  document.head.appendChild(link);
}
export function removeWoodAsh() {
  document.getElementById('QYL-WoodAsh')?.remove();
}
```

启用时动态挂入 `<link>` 标签，无需 side-effect；取消时移除即可。

### 4. 与其他主题对比

| 特征 | QYLWildness | QYLWoodAsh |
|------|------------|------------|
| 适用模式 | **暗色** `[data-theme-mode="dark"]` | **浅色** `[data-theme-mode="light"]` |
| 主色 | `#9db56d` 橄榄绿（饱和约 25%） | `#3f50a3` 靛蓝（饱和度 40%） |
| 背景色系 | 深灰绿 `#2d353b` | 暖灰米 `#dddbc7` |
| 文字色系 | 暖白/米色 | 深靛蓝/蓝灰 |
| 主色 hue | `123deg`（绿） | `105deg`（蓝-靛） |
| 毛玻璃 | `rgba(34,39,38,0.7)` | `rgba(225,225,219,0.7)` |
| 字体专属 | 无 | 无（共通用 13 级字体色系统） |
