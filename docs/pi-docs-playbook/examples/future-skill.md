# Example: Create A Future Skill

```md
I want to turn this repo into a real skill for pi-based agent application development.

Use this repo as your documentation reference. Read the relevant source files and draft:

- what the skill should trigger on
- what source files it should read first
- what it should refuse to answer without checking
- what checklist it should apply
- why skill-draft/ is not yet a finished skill
```
