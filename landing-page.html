01 · 功能

核心能力

  1. 智能对话多轮上下文感知

    右栏 AI Agent 自动读取当前 Excel 上下文——文件、工作表、列名、样例数据。你用自然语言描述需求，AI 已经知道你在操作什么。

  2. 公式生成自然语言 → Excel 公式

    描述计算逻辑，Agent 生成可用公式并一键应用。支持公式预览、历史记录、快捷收藏，{} 占位符自动按行展开。

  3. LLM 批量处理逐行调用大模型

    翻译、分类、摘要、信息提取——逐行调用 LLM 处理 Excel 数据，结果写回表格。支持暂停/恢复/中止，断点续传，实时进度与 ETA 显示。

  4. Python 脚本执行AI 编写并自动修错

    复杂的数据清洗、跨文件合并、统计汇总？用自然语言描述需求，AI 自动编写 Python 脚本并执行，出错时自动诊断重试。

  5. 提示词库模板复用

    创建、编辑、搜索提示词模板。保存管理 Agent 生成的提示词，LLM 批量页面下拉即可选用。

  6. 模型配置9 类 LLM Provider

    OpenAI、Anthropic、DeepSeek、Google、Mistral 等。每模型独立代理开关，API Key 加密存储，一键测试连接。

  7. 技能系统可扩展的 Agent 能力

    内置 Python 数据处理技能，支持从本地文件夹导入。Agent 自动发现可用技能，可视化浏览和编辑技能文件。

  8. 双模式设计直接执行 + Agent 辅助

    每个核心功能同时支持两种方式：中栏表单直接操作处理重复任务，右栏对话辅助首次生成和复杂需求。Agent 生成的结果可直接落地到中栏执行。
