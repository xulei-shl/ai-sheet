Expand description

The ChartType enum define the type of a Chart object.

The main original chart types are supported, see below.

Support for newer Excel chart types such as Treemap, Sunburst, Box and Whisker, Statistical Histogram, Waterfall, Funnel and Maps is not currently planned since the underlying structure is substantially different from the implemented chart types.

Scatter

A Scatter chart type. Scatter charts, and their variant, are the only type that have values (as opposed to categories) for their X-Axis. The default scatter chart in Excel has markers for each point but no connecting lines.

§

ScatterStraight

A Scatter chart type where the points are connected by straight lines.

§

A Scatter chart type where the points have markers and are connected by straight lines.

§

ScatterSmooth

A Scatter chart type where the points are connected by smoothed lines.

§

ScatterSmoothWithMarkers

A Scatter chart type where the points have markers and are connected by smoothed lines.

§

Stock

A Stock chart showing Open-High-Low-Close data. It is also possible to show High-Low-Close data.

Note, Volume variants of the Excel stock charts aren’t currently supported but will be in a future release.
