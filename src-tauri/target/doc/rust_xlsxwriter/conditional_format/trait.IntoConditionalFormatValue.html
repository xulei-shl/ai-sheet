Expand description

Trait to map rust types into an ConditionalFormatValue value.

The IntoConditionalFormatValue trait is used to map Rust types like strings, numbers, dates, times and formulas into a generic type that can be used to replicate Excel data types used in Conditional Formats.

See ConditionalFormatValue for more information.
